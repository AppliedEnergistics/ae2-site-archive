    storage bus export all item send in the network to their linked storage just like an export bus so you dont need it after moving all the items that where in the network before you install the storage bus :) (yep it dont export the item already in cell and in that case it will often use cell as a higher priority than the storage bus so you need to empty the cell of the items first or just remove them from the network temporaly and see the * below) i have 3 DSU connect to my network using storage bus only i can send and take from them wirthout problems from all my terminal. *on another note you can send the items into the storage bus using an io port, just configure you storage bus with the items you want in the dsu and place your cell in an io port it will extract all remaining items into the storage bus automatically :), when done replace all you storage cell and now if you add the items that are mean to go in the dsu in your network it will send them in the dsu normally. you dont really need 2 network too. if it dont work you messed something in your config of the storage bus or the me drive/chest (look for priority ;) )
